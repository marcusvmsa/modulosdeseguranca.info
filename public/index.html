Quase todos os bancos brasileiros implementam módulos de segurança, com o objetivo declarado de garantir a segurança dos clientes de serviços online. A grande verdade é que este é um sistema falho, implementado em uma tecnologia crivada de brechas de segurança, que mais causa aborrecimentos do que realmente ajuda.

Vulnerabilidades

Os módulos de segurança de bancos como Banco do Brasil, Itaú e Santander são criados com uma tecnologia chamada Java Applets. O problema é que a mesma tecnologia que é usada pelos bancos para "garantir" a sua segurança pode justamente colocar você em risco.

A última falha de segurança do Java levou o Departamento de Segurança americano a recomendar que os usuários desativassem imediatamente o Java de seus navegadores. Veja o que o especialista em segurança Adam Gowdiak declarou sobre esta mesma falha:

Nós não ousamos dizer aos usuários que é seguro voltar a usar o Java.

—Adam Gowdiak, Security Explorations

Um outro especialista declarou que o mais seguro é partir do princípio que o Java sempre será vulnerável.

Nos últimos 24 meses a tecnologia Java teve 16 atualizacões críticas e de segurança. Outras falhas de segurança podem levar até 2 anos para serem corrigidas.

Mesmo diante de todas essas recomendações, os bancos insistem em dizer que é uma medida de segurança.

Aborrecimentos

Como se não fosse suficiente os bancos exigirem que você use uma tecnologia que pode comprometer a sua segurança, ela também traz uma série de aborrecimentos aos clientes.

Instalar e configurar o seu navegador com o Java não é tarefa simples, mesmo para os mais experientes. Isso porque existem muitos fatores que podem fazer com que a instalação falhe, fazendo você perder tempo e dinheiro.

Além disso, por causa dos Java Applets, você não pode acessar os sites de bancos através de seu smartphone ou tablet -- a menos que você instale um aplicativo específico para o seu aparelho. O mais curioso é que o acesso por meio destes aplicativos é feito sem o uso de Java Applets.

O acesso feito por aplicativos para smartphones e tablets não usa a tecnologia Java e isso não faz com que seu acesso seja menos seguro (pelo contrário, dadas as falhas de segurança dos Java Applets).

Se o uso de Java Applets fosse realmente essencial, você acha que os bancos iriam criar aplicativos sem eles? Pense de novo!

Problemas com Java Applets

Por causa da última falha de segurança dos Java Applets, a Apple decidiu que seria mais seguro atualizar seu sistema operacional, desativando completamente o Java.

Por causa disso, milhares de clientes não podem mais acessar os serviços online, perdendo mais tempo e mais dinheiro. Os bancos cobram mensalmente a cesta de serviços (cobrança abusiva, mesmo com o aval do governo) que inclui o acesso ao Internet Banking. Se você não consegue configurar o seu computador com Java Applets, seja porque não sabe como fazer isso, ou porque simplesmente não é possível, está sendo lesado.

Como se proteger?

Se usar Java Applets não aumenta a segurança enquanto você acessa o site de seu banco, o que você pode fazer? A verdade é que grande parte das fraudes acontecem por descuido dos usuários.

Siga as dicas abaixo e estará muito mais seguro, não apenas no site de seu banco.

  • Nunca abra e-mails enviados por bancos. É muito fácil forjar o autor de um e-mail. Então, para que arriscar? Isto vale para e-mails que avisam sobre atualizações cadastrais de bancos e CPF/CNPJ ou irregularidades junto ao SPC/Serasa. Na dúvida, ligue para estes órgãos e solicite mais informações sobre o problema.
  • Não use a mesma senha em serviços diferentes. A maioria dos usuários usa a mesma combinação de e-mail e senha em serviços diferentes. Embora seja prático, isso deixa você mais exposto, caso algum desses serviços sofra algum tipo de invasão.
  • Use uma senha forte. Use caracteres especiais, acentos e evite palavras disponíveis em dicionários. Quanto mais difícil for a sua senha, melhor.
  • Não use computadores compartilhados para acessar o site do seu banco. Sabemos que nem todos têm computador próprio e que muitos utilizam computadores a partir de LAN houses. O problema é que você não tem como garantir que o seu acesso não está sendo monitorado.
  • Mantenha seu computador sempre atualizado. Atualize sempre o seu computador. Estas atualizações contêm correções de bugs e de falhas de segurança.

O que posso fazer?

Pelo fato de instituições bancárias sempre estarem embaixo das asas do governo, na prática não há muito o que fazer a não ser reclamar. Reclame muito.

Reclame com seu gerente. Envie mensagens para a ouvidoria do banco do qual você é cliente. Avise amigos, parentes e vizinhos para que façam o mesmo. Quem sabe assim, com muito barulho, possamos mudar alguma coisa!
